# Realtime Chat

Небольшое Rails-приложение с общим чатом на Action Cable и Turbo Streams. Интерфейс собран на Bootstrap и адаптируется под мобильные экраны.

## Запуск
1. Установите зависимости: `bundle install`.
2. Примените миграции: `bin/rails db:migrate`.
3. Запустите сервер: `bin/rails server`.
4. Откройте `http://localhost:3000` и общайтесь в одной комнате с другими подключёнными пользователями.

## Особенности
- Общий канал с автообновлением сообщений в реальном времени.
- Случайное гостевое имя сохраняется в сессии.
- Автопрокрутка к последним сообщениям и адаптивный Bootstrap-макет.
